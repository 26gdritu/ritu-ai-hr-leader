## üÜò TROUBLESHOOTING

### Website Not Loading:
1. Check GitHub Pages is enabled: Settings ‚Üí Pages ‚Üí Source: main branch
2. Wait 1-2 minutes for deployment
3. Clear browser cache
4. Try incognito/private mode

### Not Appearing in Google:
1. Check if indexed: `site:26gdritu.github.io/ritu-ai-hr-leader`
2. Check Google Search Console for errors
3. Timeline: Can take 1-7 days after requesting indexing
4. Ensure robots.txt allows crawling

### Git Push Fails:
1. Check if Personal Access Token expired
2. Generate new token: https://github.com/settings/tokens
3. Use: `git remote set-url origin https://TOKEN@github.com/26gdritu/ritu-ai-hr-leader.git`
4. Push again

### Styling Broken:
1. Check if CSS file path is correct in HTML
2. Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
3. Check browser console for errors
